# Users & Roles

Manage users and their permissions in Mist.

## User Roles

### Admin
- Full system access
- User management
- System configuration
- Access all projects

### User
- Create projects
- Deploy applications
- Invite project members
- Limited to assigned projects

## Managing Users

See [Authentication](./authentication) for user management details.

## Project Roles

Within projects, users can have different roles:

- **Owner** - Full project control
- **Admin** - Manage apps and members
- **Member** - View and deploy
